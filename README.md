# Airbnb-Global-Performance-Dashboard

🌍 Global Airbnb Performance Dashboard (Power BI)

📌 Project Overview

This project analyzes Airbnb listing and review data to understand global performance trends, market distribution, pricing behavior, customer review patterns, and host trust signals.

The dashboard was built using Power BI with custom DAX measures and calculated columns to generate business insights.

Dataset Source: Maven Analytics

🎯 Business Objectives

Analyze total listings, hosts, cities, and reviews globally

Understand market share distribution by city

Identify pricing trends by property type

Study customer review frequency behavior

Analyze rating performance across cities

Evaluate host trust signals (profile picture & identity verification)

Examine seasonality patterns in reviews

🧠 Key Insights

Paris, NYC, and Sydney account for nearly half of total listings.

Most customers leave only one review.

Mexico City & Rio show highest overall ratings.

Cleanliness and value scores are generally lower across cities.

Over two-thirds of hosts are identity verified.

Strong seasonality pattern observed during European summer months.
