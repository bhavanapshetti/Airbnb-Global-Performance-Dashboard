# Airbnb-Global-Performance-Dashboard

🌍 Global Airbnb Performance Dashboard
📌 Project Overview
This project analyzes Airbnb global listing and review data to understand market performance, pricing trends, customer behavior, and host trust signals.
The dashboard was built using Power BI with custom DAX measures.

🎯 Objectives
Analyze total listings, hosts, cities, and reviews
Understand market share distribution by city
Compare pricing by property type
Study review frequency patterns
Analyze ratings across cities
Evaluate host trust indicators
Identify seasonal trends

🛠 Tools Used
Power BI
DAX
Data Modeling
Data Cleaning
Data Visualization

📊 Key Insights
Paris, NYC, and Sydney contribute nearly half of total listings.
Most customers leave only one review.
Mexico City and Rio have highest overall ratings.
Cleanliness and value score lower compared to other metrics.
Majority of hosts are identity verified.
